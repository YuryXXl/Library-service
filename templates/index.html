üìö Library API

This API allows users to manage a list of books in a virtual library.

üìå Endpoints

GET /api/books

Returns a list of all books.

Response Example:

[{ "book_id": 1, "title": "Clean Code", "author": "Robert C. Martin" }]

GET /api/books/<book_id>

Returns a specific book by ID.

Error: 404 Book not found

POST /api/books

Add a new book. Required: title, author

Body Example:

{
  "title": "Clean Code",
  "author": "Robert C. Martin",
  "year": 2008,
  "genre": "Programming",
  "status": "to read",
  "rating": 5,
  "notes": "Must-read for devs"
}

PUT /api/books/<book_id>

Update an existing book. Fields are optional.

Error: 404 Book not found

DELETE /api/books/<book_id>

Deletes a book by ID.

Error: 404 Book not found

GET /api/books/stats

Returns total count, genre breakdown, status distribution, and average rating.

üìã Field Reference

  • title (string, required)
  • author (string, required)
  • year (integer, optional)
  • genre (string, optional)
  • status (string: "read", "reading", "to read")
  • rating (integer, 0-5)
  • notes (string, optional)
